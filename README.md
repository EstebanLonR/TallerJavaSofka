La estructura completa del projecto es la siguiente :

SourcesPackage/

Package ejercicios: contiene todos los ejercicios propuestos en el Taller de Evaluacion De Java del numeral 1 al numeral 15 y su respectiva clase ejecutable.

Package personas: contiene las clases necesarias para solucionar el ejercicio numero 16 del Taller de Evaluacion De Java, con su respectivo ejecutable.

Package electrodomesticos: contiene las clases necesarias para solucionar el numeral 17 del Taller de Evaluacion De Java, con su respectivo ejecutable.

Package seriesyvideojuegos: contiene las clases necesarias para solucionar el numeral 18 del Taller de Evaluacion De Java, con su respectivo ejecutable.
